# Contributing

## Issues report

**An issues report must include:**

- Information of version of game you are playing

- Crash log/game log

- What are you doing in the game when the problem happened

**Make sure the version of mod is updated and you don't change any code and compile and use that**

## Suggested Features

### Idea

- Explain in Issues tab

- Choose enhancement label

### Coding for suggested features

**Step 1**: Fork all branches the project and download to your computer

**Step 2**: Using MCreator to make the features you want

**Step 3**: Export workspace (In MCreator, find the gui on mcreator.net) to a shareable .zip

**Step 4**: Extract the all file, commit it to `dev` branch of the fork

**Step 5**: Open A pull request and direct to the `dev` branch of original repositories, then explain what you add

**Note**: Workspace mustn't require any plugin

## Translation

**Step 1:** Fork all branches the project

**Step 2:** **Go to:** `dev` branch and `src/main/resources/assets/moreandmoreendrods/lang`

**Step 3**: Copy the en_us.json

**Step 4**: Create your translation file in that location (like vi_vn.json) and translate. Finally, open pull request and direct it to the `dev` branch of original repositories
